Tavern's Gear 

Интернет-магазин комплектующих для настольной игры Dungeons & Dragons.

Реализованный функционал

Для всех пользователей:
- Просмотр каталога товаров
- Фильтрация товаров по категориям
- Сортировка товаров (по новизне, названию, цене, году выпуска)
- Страница "О нас"

Регистрация и авторизация:
- Форма регистрации с валидацией:
  - Имя и фамилия (только кириллица)
  - Логин (латиница, цифры, дефис)
  - Email (уникальный)
  - Пароль (минимум 6 символов)
  - Подтверждение пароля
  - Согласие с правилами
- Форма входа с сообщениями об ошибках на русском языке
- Выход из системы

Функционал клиента:
- Добавление товаров в корзину (на странице каталога и товара)
- Управление корзиной (изменение количества, удаление)


Админ-панель:
- Управление категориями товаров (добавление, редактирование)
- Управление товарами (добавление, редактирование, удаление)
- Просмотр списка пользователей


Технологии

- **Django 3.2** - веб-фреймворк
- **SQLite** - база данных
- **HTML/CSS** - верстка
- **JavaScript** - интерактивность
- **Git** - система контроля версий

Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/SafiaNR/Taverns-gear.git
cd Taverns-gear
Создайте виртуальное окружение:

bash
python -m venv venv
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
Установите зависимости:

bash
pip install -r requirements.txt
Выполните миграции:

bash
python manage.py makemigrations
python manage.py migrate
Создайте суперпользователя:

bash
python manage.py createsuperuser
Запустите сервер:

bash
python manage.py runserver
Откройте в браузере: http://127.0.0.1:8000/

Доступ к админ-панели
Админ-панель доступна по адресу: http://127.0.0.1:8000/admin/
